# Time-Locked Savings Bank

A Solidity smart contract that simulates a decentralized savings vault with time-locked deposits and secure ETH withdrawals.
---------------------------------------------------
# Features

Time-locked ETH deposits
Partial withdrawals
Secure ETH transfers using .call
Event logging
Custom errors
Reentrancy-safe withdrawal flow
-------------------------------------------------
# Concepts Used

Solidity smart contracts
Structs & mappings
block.timestamp
Events
Custom errors
CEI security pattern
--------------------------------------------------
# Run in Remix

Open Remix IDE
Create TimeLockedVault.sol
Paste contract code
Compile with Solidity ^0.8.x
Deploy with MetaMask
